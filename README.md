# Messify 🍽️

> Track meals. Split fair. Stay stress-free.

A Pinterest-inspired mess management web app built with Next.js, Firebase, and Tailwind CSS.

## Features

- 🔥 Real-time data sync via Firebase Firestore
- 📱 Pinterest-style masonry card layout
- 👥 Multi-mess support with role management
- 🧾 PDF report generation (jsPDF)
- 📴 Offline support with IndexedDB persistence
- 🔐 Google & Email/Password auth

## Setup

1. Clone the repo
2. Copy `.env.example` → `.env.local` and fill in Firebase credentials
3. `npm install`
4. `npm run dev`

See `guide/` folder for full setup instructions.

## Tech Stack

- **Next.js 14** (App Router)
- **Firebase** (Auth + Firestore)
- **Tailwind CSS**
- **jsPDF + autoTable**
